# Marketplace Web

Este é o meu projeto executado seguindo as instruções do desafio prático de Marketplace Web. Aqui, coloquei em prática diversos conceitos e tecnologias para desenvolver um dashboard de gestão de vendas.

## O Projeto

O projeto consiste em um dashboard para produtores, com as seguintes funcionalidades:

- **Cadastro de usuários**
- **Upload de imagens**
- **Autenticação**
- **Dashboard com métricas:**  
  - Produtos vendidos  
  - Produtos anunciados  
  - Visualizações do produto  
  - Gráfico de visualizações por dia
- **Listagem de produtos** com informações detalhadas (imagem, status, categoria, título, preço e descrição)
- **Filtros de busca**: pesquisa por texto e filtro por status, com query params na URL
- **CRUD de produtos:**  
  - Criação e edição de produtos (com restrições para produtos vendidos ou desabilitados)
  - Marcar/desmarcar produtos como vendidos ou desabilitados
- **Logout da aplicação**

A API utilizada já vem com dados pré-semeados para facilitar os testes.
